# Lost and Found Item Management System

## 📌 Project Description
This is a web-based application to manage lost and found items on a campus.

## 🚀 Features
- User Registration & Login
- Profile System (Name + Image)
- Post Lost/Found Items
- Global Search with Filters
- Chat System between users

## 🛠 Tech Stack
- Python (Flask)
- MongoDB
- HTML, CSS

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
